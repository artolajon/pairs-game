Pairs Game

Try to hide all the numbers

Game basics

There is a list of random number that you need to clear.

How to clear

To clear the list you need to find cells with the same number.

The cells need to be one next to the other.

Valid moves

  • Forward, backward, up or down
  • Ignoring hidden ones
  • Using the next row

Once there aren´t more pairs

Once there aren´t more pairs, you can add more numbers pressing the "" button

When does it finish?

Once you cleared all the list
